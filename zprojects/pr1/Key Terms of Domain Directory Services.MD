Domain - A unique name that identifies a website on the internet.

Directory Services - A service that stores and organizes information about domains, their associated IP addresses and other related information.

DNS (Domain Name System) - A hierarchical and distributed naming system for computers, services or any resource connected to the Internet or a private network.

IP (Internet Protocol) Address - A unique numerical label assigned to each device connected to a network that uses the internet protocol for communication.

Domain Registrar - An organization accredited by ICANN (Internet Corporation for Assigned Names and Numbers) or a national ccTLD (country code top-level domain) authority to manage the registration of domain names.

WHOIS - A protocol used to query databases that contain information about registered domain names and their associated IP addresses, including the name, address, phone number, and email address of the registrant.

DNSSEC (DNS Security Extensions) - A set of protocols that enhance the security of the DNS by adding digital signatures to domain name records.
